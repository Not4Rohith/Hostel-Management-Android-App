# Hostel Management App

## Overview
This project was developed as part of a college Skill Development Program.  
The application helps manage hostel-related tasks such as student records,
room allocation, and administrative management.

## Features
- Manage student details
- Room allocation tracking
- Basic hostel administration interface

## Tech Stack
- TypeScript
- JavaScript (Node.js)
- SQLite
